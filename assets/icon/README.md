# Icône de l'application

Place ici le logo de Correctis sous le nom **`app_icon.png`**.

Recommandations :
- Format : PNG carré
- Taille : 1024 × 1024 px (idéal)
- Le fichier doit s'appeler exactement `app_icon.png`

Une fois le fichier en place, génère les icônes :

```
flutter pub get
dart run flutter_launcher_icons
```

Cela génère automatiquement toutes les tailles d'icônes pour Android et iOS.
